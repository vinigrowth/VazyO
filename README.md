## O que é

VazyO é um sistema operacional que roda dentro do Claude Code.
Você instala, configura uma vez e ganha comandos prontos pra criar páginas de vendas, sites, e-books, carrosséis e anúncios — com UI/UX profissional, copy estratégica e deploy automático.

Cada entrega é única. Nenhum site parece com o outro. Nada tem cara de IA.

---

## Instalação

```bash
git clone https://github.com/vinigrowth/VazyO.git
cd VazyO
code .
```

No VS Code, abra o terminal integrado → digite `claude` → rode `/instalar`.

O sistema vai te entrevistar, montar a memória do seu negócio e deixar tudo pronto.

---

## Comandos

| Comando | O que entrega |
|---|---|
| `/instalar` | Setup inicial — roda uma vez só |
| `/abrir` | Carrega contexto antes de trabalhar |
| `/salvar` | Commit + push no GitHub |
| `/atualizar` | Sincroniza memória com o que mudou |
| `/novo-projeto` | Cria pasta isolada por cliente |
| `/pagina-de-vendas` | Página completa com copy pra público frio + deploy |
| `/site` | Site profissional por nicho + deploy |
| `/ebook` | E-book completo em HTML elegante + preview |
| `/carrossel` | Carrossel Instagram 1080x1350 com identidade |
| `/seo` | Fluxo completo de otimização |
| `/anuncio` | Copy e estrutura pra Meta Ads e Google Ads |

---

## Como funciona

```
_memoria/     →  contexto do negócio (lido a cada sessão)
identidade/   →  visual da marca (aplicado em tudo)
projetos/     →  um subpasta por cliente
saidas/       →  tudo que o sistema gera
.claude/skills/ →  os comandos
```
